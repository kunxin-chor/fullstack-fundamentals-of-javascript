Write a function, **posOrNeg**, that takes an integer **argument.**

If the integer is positive, returns the string **"+"**, or else returns the string **"-"**  If the number is 0, just return string "0"

```
# Example Input
foo = posOrNeg(12)

# Correct Output
+
```

```
# Example Input
foo = posOrNeg(-12)

# Correct Output
-
```

```
# Example Input
foo = posOrNeg(0)

# Correct Output
0
```
