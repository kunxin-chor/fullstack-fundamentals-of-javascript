Write a function named sayHello that asks the user for two inputs.

The two inputs are for variables firstName and lastName.

The function will print out "Hello, `<firstName>` `<lastName>`"._

** Test Cases **
```
> John
> 0
Hello, John O
```
```
> Charlie
> Bronze
Hello, Charlie Bronze
```
```
> Megan
> Batoon
Hello, Megan Batoon
```
